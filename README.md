# Customer Value Analysis

## Overview

This project analyzes customer purchase behavior for a direct-to-consumer (D2C) fashion brand using Python, SQL, and Power BI. The objective is to identify high-value customers, understand promotional dependency, and develop data-driven customer retention strategies.

## Objectives

- Identify high-value customer segments
- Measure promotional dependency
- Engineer customer value metrics
- Develop SQL-based business insights
- Build an interactive Power BI dashboard
- Recommend retention strategies

## Tech Stack

- Python (Pandas, NumPy)
- SQL
- Power BI
- Google Colab

## Project Workflow

```
Raw Dataset
      ↓
Python
(Data Cleaning + Feature Engineering)
      ↓
SQL
(Customer Segmentation & Business Analysis)
      ↓

(Interactive Dashboard)
      ↓
Business Recommendations
```
